# Snapshots

Generated browser screenshots, DOM captures, and runtime screen captures used during audits/research.

These files are useful locally but can be large and may accidentally include private UI state, so they are ignored by Git by default.

Subfolders:

- `thinkmay_screenshots/` - Thinkmay website screenshots captured during brand/UX review.
- `thinkmay_dom/` - Captured DOM/HTML snapshots.
- `runtime/` - Local runtime/screen captures.
