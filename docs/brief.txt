Thinkmay CloudPC is a cloud pc service. It 
provides a virtual desktop environment that can be accessed from any device with an internet connection. 
It is a secure and reliable way to access your work environment, and it is a great way to stay connected with your colleagues. 

Features:
- Video codec support h264/h265
- Microphone support
- Mobile / desktop compatible
- Gamepad support
- Virtual gamepad support
- Adaptive bitrate (powered by google congestion control)
- Multi routing (high availability network)
- GPU accelerated video encoding
- Use WebRTC for streaming (optionally FlexFEC/ NACK + RTX supported)
- Touch / multi touch supported

Supported  and well tested browser:
- Chrome
- Safari

Server location:
- HCM (Ho Chi Minh City)
- HP (Hai Phong)
