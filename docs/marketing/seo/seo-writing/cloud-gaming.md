- **Gaming with friends tonight?** Start a session without building or upgrading hardware.

## Frequently Asked Questions

### What is cloud gaming?

Cloud gaming runs the game on a remote machine and streams the video to your device. Your controls are sent back over the internet, so even a low-spec phone or laptop can play games that require stronger hardware.

### Do I need to own the games?

Yes. Thinkmay provides the Cloud PC hardware and streaming platform. You need to own paid games on Steam, Epic Games, EA App, Rockstar, Garena, or other publisher accounts.

### What internet speed do I need?

For 1080p gameplay, a stable 15-20 Mbps connection is a good starting point. A 5GHz Wi-Fi network or wired Ethernet connection usually performs better than crowded 2.4GHz Wi-Fi.

### Can I use a controller?

Yes. Thinkmay supports browser gamepad input when your device and browser expose the controller correctly. Mobile users can also use the built-in virtual gamepad overlay.

### Is Thinkmay only for gaming?

No. The same Windows 11 Cloud PC can be used for productivity, development, 3D tools, and Windows-only software.
