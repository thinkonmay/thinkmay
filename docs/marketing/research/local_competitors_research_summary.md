# Local Competitor Research Summary

_Date: 2026-05-21_

## Scope

This summary reviews the local Vietnam competitors the user identified:

- RevoltPC / RevoltG
- EZYCLOUDX
- PCRender
- TrumBox

The goal is to understand how these brands position themselves and how competitive they appear in organic search relative to Thinkmay.

## Executive summary

Among the identified local competitors, **Revolt** appears to be the strongest in SEO value and brand-driven search demand. **EZYCLOUDX** is the strongest utility-style competitor for GPU rental and some cloud PC terms. **PCRender** is a smaller but relevant cloud PC / rental machine competitor. **TrumBox** is currently the least clear and appears to have little measurable organic visibility.

Thinkmay remains strong in Vietnam-local cloud gaming and cloud PC intent, but the local market appears to split across distinct SEO lanes:

1. gaming-first branded demand
2. GPU rental / AI / rendering utility demand
3. cloud PC rental / server-style demand
4. experience-first browser cloud gaming / cloud PC demand

## Competitor profiles

### 1. RevoltPC / RevoltG
**Likely domain:** `revolt.vn`

**What it is**
- Cloud PC / cloud gaming platform
- Gaming-first positioning
- Strong focus on licensed game access and “high-spec” game experience

**Observed messaging**
- “Nền tảng trải nghiệm game bản quyền”
- “Giải pháp hạ tầng Cloud PC cấu hình cao”
- “Phần mềm cho thuê PC”

**Audience**
- Gamers who want to access licensed game platforms and use a cloud PC optimized for gaming

**SEO observations**
- Strongest measured local competitor by estimated organic traffic value
- Approximate Vietnam organic footprint:
  - **116** ranking keywords
  - **30,122** estimated traffic value
- Strength seems heavily tied to brand demand and gaming-related branded search terms

**Notable keyword themes**
- `revoltg`
- `revoltg pc`
- `revolt`
- `choi game`

**Assessment**
Revolt is a serious local competitor. Its SEO strength appears to come more from **brand strength and gaming identity** than from a wide range of generic cloud infrastructure terms.

### 2. EZYCLOUDX
**Domain:** `ezycloudx.com`

**What it is**
- GPU cloud / GPU rental platform

**Observed messaging**
- “Thuê GPU cloud giá rẻ tại Việt Nam”
- hourly billing
- AI / Machine Learning / Deep Learning / 3D Rendering / Cloud Gaming

**Audience**
- AI and ML users
- creators / render users
- people needing temporary GPU access
- some cloud gaming users

**Pricing cues**
- from about **5.000đ/giờ** for some offerings

**SEO observations**
- Smaller overall footprint than Thinkmay and Revolt
- Approximate Vietnam organic footprint:
  - **21** ranking keywords
  - **1,924** estimated traffic value
- Very strong on some transactional GPU/cloud-PC terms

**Notable keyword themes**
- `thuê gpu theo giờ` → strong ranking
- `cloud pc` → strong ranking
- `cloudpc` → strong ranking
- GPU rental and GPU cloud intent

**Assessment**
EZYCLOUDX is not the closest pure cloud-gaming competitor, but it is a meaningful **cloud PC / GPU / utility search competitor**. It overlaps with Thinkmay where users are searching for remote compute or rent-by-the-hour performance.

## What this means for Thinkmay

### Main local threats
The most meaningful local SEO competitors are:

1. **Revolt** for gaming-first branded demand
2. **EZYCLOUDX** for cloud PC / GPU utility intent
3. **PCRender** for machine-rental / cloud-PC rental intent

### Strategic read
- **Revolt** is the strongest local competitor in measured SEO value
- **EZYCLOUDX** is the strongest utility/infrastructure-style keyword competitor
- **PCRender** is the clearest long-tail rental competitor
- **TrumBox** is not currently a major visible organic threat based on the available evidence

### Implication for Thinkmay
Thinkmay is still strong on Vietnam-local cloud gaming and cloud PC demand, but the market is segmented. Competitors are not all fighting on the same message:

- Revolt competes on **brand + gaming legitimacy**
- EZYCLOUDX competes on **GPU utility + price/performance**
- PCRender competes on **rental machine practicality**
- Thinkmay competes on **experience-first cloud gaming / browser cloud PC**

## Recommended SEO strategy updates for Thinkmay

Yes, Thinkmay should update its SEO strategy, but the change should be an **adjustment rather than a full overhaul**.

### What to keep doing

Thinkmay should continue defending the intent it already owns well:

- Vietnam-local cloud gaming queries
- Vietnam-local cloud PC queries
- browser-based gaming positioning
- high-performance gaming + work positioning
- mobile and low-end-device accessibility messaging

These are already strong parts of Thinkmay's market fit and should remain the center of the brand.

### What to change

#### 1. Defend against Revolt on branded gaming trust

Revolt appears strongest where gaming identity and brand demand matter most. Thinkmay should respond by strengthening trust and gaming legitimacy content, such as:

- comparison pages such as **Thinkmay vs Revolt**
- pages focused on supported games, game launchers, and game libraries
- customer proof, testimonials, creator/community mentions, and gameplay examples
- stronger pages around “chơi game AAA”, “cloud gaming Việt Nam”, and similar gamer-first commercial queries

#### 2. Differentiate from EZYCLOUDX and PCRender instead of copying them directly

EZYCLOUDX and PCRender compete more on utility, rental, and infrastructure-style language. Thinkmay should not fully shift into a generic GPU-rental message unless that is the actual business strategy.

Instead, Thinkmay should clearly explain how its offer differs:

- easier than raw GPU rental
- better for gamers who do not want setup complexity
- browser-first, no-download experience
- ready-to-use cloud PC rather than generic compute capacity
- stronger mobile / controller / consumer experience

Useful page ideas:

- **Cloud PC chơi game vs thuê GPU**
- **Thuê Cloud PC theo giờ để chơi game**
- **Cloud PC cho game thủ không cần cài đặt phức tạp**

#### 3. Expand page coverage around intent clusters already visible in the market

The local landscape suggests three practical clusters worth targeting more deliberately:

- **gaming-first intent**
- **cloud PC / rental intent**
- **productivity / rendering / work intent**

That means Thinkmay should create or strengthen dedicated pages for:

- cloud gaming Vietnam
- cloud PC Vietnam
- cloud PC gaming
- thuê cloud PC
- cloud PC cho gaming
- cloud PC cho render 3D
- cloud PC cho làm việc từ xa
- chơi game trên trình duyệt
- chơi game AAA không cần tải xuống

#### 4. Separate product narratives by use case

Competitors are splitting the market by message. Thinkmay should make that segmentation clearer in its own content structure.

Recommended narrative split:

- **Gaming** — AAA games, browser play, low latency, controller/mobile support
- **Creative work** — render, 3D, Adobe, workstation replacement
- **Remote work / general compute** — access anywhere, no expensive hardware, persistent desktop experience

#### 5. Deprioritize low-fit GPU / AI rental SEO unless Thinkmay truly wants that audience

EZYCLOUDX is strong on GPU-rental intent. That does not automatically mean Thinkmay should chase all those queries.

Thinkmay should only target deep GPU/AI rental terms if:

- the product truly supports that workflow well
- the landing pages can convert those users
- the business wants to serve that audience long term

Otherwise, chasing those terms may dilute the brand's stronger cloud gaming / consumer cloud PC identity.

### Recommended priority order

#### High priority
- defend and expand Vietnam cloud gaming / cloud PC landing pages
- publish comparison content against Revolt and similar local alternatives
- create clearer pages for gaming, rendering, and remote work use cases
- strengthen commercial pages around “cloud PC chơi game”, “thuê cloud PC”, and browser gaming

#### Medium priority
- build educational content around latency, mobile play, browser compatibility, and no-download gaming
- add pages clarifying the difference between Thinkmay and generic GPU/server rental

#### Lower priority
- broad AI/GPU rental keyword campaigns
- international expansion pages not tied to real go-to-market plans
- generic infrastructure-style content that does not match Thinkmay's core experience-first positioning

## Conclusion

The local competitive landscape is not dominated by a single type of player. Instead, different competitors win different intent clusters:

- **Revolt** wins the strongest branded gaming demand
- **EZYCLOUDX** wins GPU-rental and some generic cloud-PC utility intent
- **PCRender** wins some cloud PC / gaming machine rental terms
- **Thinkmay** remains strong in local cloud gaming and cloud PC discovery, especially where the message is broader than GPU rental alone

If Thinkmay wants to strengthen its local moat further, it should continue defending high-intent Vietnam cloud gaming and cloud PC terms while deciding whether to compete more directly on:

- branded gaming demand like Revolt
- utility / GPU demand like EZYCLOUDX
- rental / server-style demand like PCRender
