## 1. Executive Summary

Thinkmay competes across three overlapping markets, not one single category:

1. **Cloud game catalog services** — users stream a managed library or supported games.  
   Main examples: **NVIDIA GeForce NOW**, **Boosteroid**, **Xbox Cloud Gaming**, **Blacknut**.

2. **Full cloud PC / cloud desktop services** — users access a Windows/cloud desktop and can install their own games/apps.  
   Main examples: **Shadow PC**, **airgpu**, **Loudplay**, **Vagon**, **Paperspace Core**.

3. **Local GPU rental / GPU workstation providers** — users rent GPU capacity for AI, rendering, learning, gaming, or experimentation.  
   Main example in this analysis: **EZYCLOUDX**.

Thinkmay’s strongest defensible position is not “best cloud gaming service” in the global generic sense. NVIDIA, Xbox, and Boosteroid own much of that language at global scale. Thinkmay’s more differentiated position is:

> **A Vietnam/SEA-first cloud internet cafe: a full Cloud PC for gamers and students who want to play, work, install software, and use a powerful PC from old laptops, phones, tablets, Macs, or TVs without buying hardware.**

The competitor landscape shows a clear opportunity: most global players are either **catalog-first** and less flexible, or **workstation-first** and too expensive/technical. Thinkmay can win by being **local, affordable, gamer-friendly, community-led, and clearer about what works/does not work.**

## 4. Competitor Profiles

## 4.1 NVIDIA GeForce NOW

**Website captured:** `screenshots\nvidia_geforce_now.png`  
**Public positioning observed:** “The Next Generation in Cloud Gaming,” “Your Games, Your Devices, RTX On.”

### Customer-facing promise

NVIDIA presents GeForce NOW as a premium cloud gaming platform that connects to digital PC game stores so users can stream games they already own. Public copy emphasizes:

- **1500+ games** on the homepage capture/fetch.
- Connection to digital PC game stores.
- Support across laptop, desktop, Mac, TV, Android, iPhone, and iPad.
- Premium GeForce RTX rigs.
- High resolution, ultra-low latency, competitive experience.
- No game downloads, updates, or patches.

The membership page also emphasized:

- Premium members-only features.
- Install-to-Play for Performance and Ultimate members.
- Performance/Ultimate day passes.
- 50-Series performance, DLSS 4, up to 5K/120 and 360 FPS claims for Ultimate messaging.

### Product model

GeForce NOW is primarily a **catalog/store-linked cloud gaming service**, not a general-purpose personal Windows Cloud PC. Users can play supported games connected to their game store accounts, but they do not receive broad install-anything PC ownership like Shadow/airgpu/Thinkmay.

### Strengths

- Huge NVIDIA brand trust and GPU credibility.
- Strong product clarity: any device, no downloads, RTX, supported games.
- Strong trust signals: ecosystem, partners, official apps, broad device support.
- Clear category ownership in “cloud gaming.”
- Premium performance halo from RTX/DLSS language.

### Weaknesses / openings for Thinkmay

- Library/catalog restrictions remain a key difference from a full Cloud PC.
- Less local/community intimacy for Vietnam/SEA users.
- Less “internet cafe” feeling; more global platform than local helper.
- Users may need to already own supported games on connected stores.
- The site language is polished but broad; Thinkmay can be more concrete and local.

### Implications for Thinkmay

Thinkmay should not try to out-NVIDIA NVIDIA on “next generation cloud gaming.” Instead:

- Use NVIDIA as the comparison anchor for **what catalog cloud gaming cannot do**.
- Emphasize “install your own games, mods, apps, tools, and files on a full Cloud PC.”
- Avoid unsupported claims like “best” or “zero latency”; NVIDIA’s credibility makes vague superlatives look weak.
- Add a comparison section: “Cloud game library vs full Cloud PC.”

## 4.2 Shadow PC

**Website captured:** `screenshots\shadow_pc.png`  
**Public positioning observed:** “The best Cloud Gaming PC,” “Turn any screen into an RTX gaming PC,” and “Your Windows PC in the Cloud.”

### Customer-facing promise

Shadow is the closest global conceptual competitor to Thinkmay. Its website directly explains the distinction between classic cloud gaming and a full PC:

- Shadow is a **full Windows PC** streamed with ultra-low latency.
- Users can install games from Steam, Epic, Battle.net, GOG, Ubisoft Connect, etc.
- Users can run Windows apps, mods, and tools.
- Any-device support includes Windows, Linux, Apple computers, smartphones, tablets, Android TV, and browser.
- No playtime limit.
- 1 Gbps bandwidth is included.
- Positioning strongly targets Mac users who want Windows gaming.

Shadow’s FAQ is unusually useful because it is honest about compatibility:

- It says a handful of games are not compatible due to anti-cheat rules.
- It says publisher policies, anti-cheat, and DRM may regulate usage on virtual machines.
- It says users outside supported countries can subscribe but quality is not guaranteed due to distance and ISPs.
- It recommends a stable internet connection of 10 Mbps or more.

### Pricing observed

The general gamer pricing page was not the one successfully fetched during this session, but the Shadow Pro/offers page showed:

- **Neo Pro:** `$44.99/month`
  - RTX 4060 16GB-class / RTX 2000 ADA 16GB-class
  - AMD EPYC 8 vCores
  - 16GB RAM
  - 1TB SSD
  - Windows 11 Home/Enterprise
  - 1Gbps / 100Mbps
- **Power Pro:** `$59.99/month`
  - RTX 3070 Ti 20GB-class / RTX A4500 20GB-class
  - AMD EPYC 8 vCores
  - 28GB RAM
  - 1TB SSD
  - Windows 11 Home/Enterprise
  - 1Gbps / 100Mbps

### Strengths

- Very clear “full Windows PC” education.
- Strong answer to catalog restrictions.
- Strong Mac-gaming and mods narrative.
- Good transparency around anti-cheat and region limitations.
- Mature website with deep FAQs, testimonials, and business/pro segmentation.

### Weaknesses / openings for Thinkmay

- Pricing is high for many Vietnam/SEA consumer segments.
- Brand is global/Western; local payment/support/community may be weaker for Vietnam users.
- Some regions have quality limitations.
- Less “budget gamer internet cafe” positioning.

### Implications for Thinkmay

Shadow is the best model for how Thinkmay should explain itself. Thinkmay should borrow the clarity, not the exact premium tone:

- “A full Cloud PC, not just a game catalog.”
- “Install Steam/Epic/Battle.net, mods, school tools, AI tools, editing apps.”
- “Some anti-cheat games may not work — check the compatibility list or try first.”
- “Best experience depends on region/network; here are Vietnam/SEA latency tips.”

Thinkmay can beat Shadow locally on price, Vietnamese support, Discord community, and SEA-specific positioning.

## 4.4 airgpu

**Website captured:** `screenshots\airgpu.png`  
**Public positioning observed:** “Your Cloud PC for Gaming, Rendering, Video Editing, VR and more.”

### Customer-facing promise

airgpu is one of the closest product-model competitors because it explicitly sells a Cloud PC for both gaming and GPU workloads:

- Play latest games and run GPU-heavy workloads from any device.
- Run GPU-intensive applications without buying new hardware.
- Connect from laptop, tablet, or phone.
- High performance up to 4K.
- Low latency via a data center close to the user.
- Install all games and software; no limited list of supported games.
- Works through Parsec or Moonlight.
- Recommends at least 10Mbit/s download and 2Mbit/s upload.
- Supports Windows, Linux, macOS, Android, and ChromeOS devices.

### Pricing observed

airgpu uses pay-as-you-go credit pricing with no subscription surprise bill:

- NVIDIA Tesla T4, similar to GTX 1080: from `$0.65/hour`.
- NVIDIA L4, similar to RTX 4060: from `$0.90/hour`.
- NVIDIA A10G, similar to RTX 3080: from `$1.05/hour`.
- NVIDIA L40S, similar to RTX 4080: from `$1.20/hour`.
- RTX series in North America:
  - Quadro RTX 4000: `$0.65/hour`.
  - RTX A4000: `$1.10/hour`.
  - RTX A5000: `$1.55/hour`.
- Storage: `$3.50/month per 50GB SSD`, available from 100GB.
- All prices exclude VAT.

### Compatibility/trust notes

airgpu is refreshingly direct:

- No free trial at the time of fetch.
- Refunds can be requested for remaining credit.
- Most games work, but some multiplayer games use anti-cheat that blocks Cloud PCs.
- Maintains a list of games with known issues.
- Mining is not allowed.
- Server locations include North America, Europe, Asia Pacific including Hong Kong, Jakarta, Mumbai, Seoul, Singapore, Sydney, Tokyo, plus Middle East and Africa.

### Strengths

- Clear “Cloud PC” and install-anything positioning.
- Transparent hourly pricing.
- Strong global region coverage, including SEA-relevant locations like Jakarta and Singapore.
- Honest anti-cheat disclosure.
- Good for mixed gaming + productivity workloads.

### Weaknesses / openings for Thinkmay

- Hourly pricing plus storage fees may feel technical/uncertain for casual gamers.
- Requires users to understand Parsec/Moonlight and manage a Cloud PC more directly.
- No free trial, only refund path.
- Less community/handholding than a local Discord-led service.
- Pricing in USD/VAT can be less friendly for Vietnam/Indonesia audiences.

### Implications for Thinkmay

airgpu is the cleanest pricing benchmark for Thinkmay’s “hourly/flexible” side. Thinkmay can differentiate by:

- Vietnamese/SEA payment methods and local currency.
- More guided onboarding and support.
- Local Discord community and game setup help.
- Bundled plans that feel simpler than pure hourly compute.
- Clearer “internet cafe in the cloud” framing rather than compute marketplace framing.

## 5. Thinkmay Baseline vs Competitors

Thinkmay’s public/local copy currently emphasizes:

- “Thinkmay Cloud PC - Best Cloud Gaming Service.”
- “Play AAA games and work on Cloud PC with RTX 5060Ti performance.”
- No downloads required.
- Works across Laptop, Mac, Mobile.
- “Thinkmay CloudPC - Your Virtual Gaming Powerhouse.”
- “Upto 5060ti, 24GB RAM.”
- 5-hour trial plan with unlimited days.
- Standard, Performance, Stable, hourly/monthly/weekly/usage concepts.
- Community trust claim: “Over 100,000+ gamers and professional users have joined.”
- Discord public server evidence from web research: 11,606 members and 627 online at fetch time.
- Public Discord description: “Mọi thiết bị ở mọi cấu hình đều có thể chiến game xịn và làm đồ hoạ, chỉ với Cloud PC và Internet.”

### 5.1 Where Thinkmay is already strong

- **Local SEA/Vietnam story:** Founder/press/community gives Thinkmay a more emotionally credible local narrative than most global competitors.
- **Full Cloud PC freedom:** This is the main product-model advantage over NVIDIA/Boosteroid/Xbox/Blacknut.
- **Gaming + work + AI/student breadth:** Broader than pure catalog gaming.
- **Community:** Discord footprint is a real trust signal if surfaced cleanly.
- **Low-cost entry:** “from $2” and a 5-hour trial are attractive if explained clearly.
- **Mobile-first SEA relevance:** The local “quán net trên mây / warnet di cloud” idea is more distinctive than generic cloud gaming.

### 5.2 Where Thinkmay is weaker than competitors

- **Homepage clarity:** NVIDIA, Boosteroid, Shadow, and airgpu each explain the product model faster.
- **Trust architecture:** Competitors show stronger policy, partner, device, compatibility, FAQ, and support structures.
- **Compatibility honesty:** Shadow and airgpu directly discuss anti-cheat/VM limitations. Thinkmay should do the same.
- **Pricing explanation:** Vagon/airgpu are clearer about hourly pricing; Boosteroid is clearer about monthly pricing.
- **Device support presentation:** Boosteroid/NVIDIA make supported devices/apps obvious. Thinkmay should make Windows/Mac/iOS/Android/browser/TV support visually explicit.
- **Proof quality:** Claims like “best,” “zero latency,” or “100,000+” need stronger evidence or softer wording.
- **Social/support execution:** Prior brand research found placeholder support links and fragmented social handles.

## 7. Pricing and Packaging Lessons

### 7.1 What competitors teach

- **Boosteroid:** Simple subscription anchor wins attention: “from €7.49/month.”
- **airgpu:** Hourly pricing is credible when GPU class, CPU/RAM, storage, and regions are transparent.
- **Shadow:** Premium monthly pricing works when positioned as a full Windows PC and includes strong specs/storage.
- **Vagon:** Calculator-style pricing helps professionals, but may be too much for gamers.
- **Paperspace:** GPU infrastructure pricing is transparent but intimidating for consumers.
- **Blacknut/Xbox:** Subscription catalog services reduce mental load but sacrifice freedom.

### 7.2 Recommended Thinkmay packaging architecture

Thinkmay should avoid making the customer choose between too many compute abstractions. Recommended structure:

1. **Try / Experience**
   - Clear 5-hour trial.
   - Explain whether it is free, paid, how to claim it, and what limits apply.
   - Show “best for testing your network/device.”

2. **Play / Standard**
   - Best for most gamers.
   - Show GPU/CPU/RAM/storage.
   - Show monthly/weekly price and any usage/session limits.

3. **Performance**
   - Best for AAA/high settings/rendering/AI study.
   - Show RTX 5060Ti/24GB RAM if current and consistently available.
   - Show why to upgrade.

4. **Stable / 24-7**
   - Best for work, always-on tasks, long sessions.
   - Rename if “Stable” is confusing; “Always-On” or “24/7 Workstation” is clearer.

5. **Add-ons**
   - Storage.
   - Game account/library access if offered.
   - Priority queue/support.

## 8. Website UX and Visual Positioning Lessons

### 8.1 Competitor UX patterns worth copying

- **Hero line must explain the product in one sentence.**
  - Shadow: full Windows PC in the cloud.
  - airgpu: Cloud PC for gaming/rendering/video editing/VR.
  - Boosteroid: run PC games on any device, no downloads, browser.

- **Device support should be visual.**
  - Boosteroid and NVIDIA make device availability feel obvious.
  - Thinkmay should show icons for Windows, Mac, Android, iOS, browser, TV if supported.

- **Three-step onboarding works.**
  - Sign up → choose plan → connect/play.
  - For Thinkmay: create account → choose Cloud PC plan → open browser/app and play.

- **Compatibility honesty builds trust.**
  - Shadow and airgpu are direct about anti-cheat and region/network constraints.

- **Pricing needs fewer surprises.**
  - If there are storage fees, usage limits, queues, or game account fees, surface them early.

### 8.2 Thinkmay website risks seen through competitor lens

- “Best Cloud Gaming Service” is generic and less credible than a concrete local claim.
- “Upto 5060ti” has a typo and should become “Up to RTX 5060 Ti.”
- If “100,000+ users” remains visible, it needs proof/context or softer wording.
- “Virtually zero latency” testimonial language is risky; better: “low latency on stable connections.”
- The strongest differentiator — full Cloud PC freedom — should be above the fold.
- Vietnamese/local trust assets should be more visible than generic English superlatives.

- Shadow-like explanation of full PC freedom.
- Boosteroid-like simplicity.
- airgpu-like pricing/compatibility transparency.

If Thinkmay fixes the homepage message, pricing clarity, trust links, and compatibility honesty, it can occupy a more distinctive and believable position than “Best Cloud Gaming Service.”
