# Competitor Analysis

Competitor research artifacts for Thinkmay Cloud PC / cloud gaming positioning.

- `thinkmay_cloudpc_competitor_analysis.md` - written competitor analysis report.
- `screenshots/` - generated competitor screenshots and capture logs.

Screenshots/logs are ignored by Git by default because they are generated, large, and may change often. Commit only curated written analysis unless there is a deliberate reason to publish a specific image.
